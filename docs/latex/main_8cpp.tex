\doxysubsection{Detailed Description}
ESP32 Webserver med Web\+Socket, SPIFFS og touchsensor-\/integration. 

Dette program implementerer en ESP32-\/baseret webserver med\+:
\begin{DoxyItemize}
\item Web\+Socket-\/kommunikation
\item Berøringssensor til tælling af events
\item SPIFFS til filhåndtering af Wi-\/\+Fi-\/konfiguration og logning
\item Dynamisk konfiguration via et Access Point
\item LED-\/kontrol og reset-\/knap med forsinkelse 
\end{DoxyItemize}
